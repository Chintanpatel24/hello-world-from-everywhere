# 🌍 Hello World in Every Programming Language

A comprehensive collection of "Hello, World!" programs written in every programming language across the world.

## 📖 About This Repository

This repository contains "Hello, World!" programs in **330+ programming languages**, frameworks, markup languages, and data formats. It's a celebration of the diversity in the programming world and serves as a reference for anyone interested in exploring different programming languages.

## 🎯 Purpose

- **Educational**: Learn the basic syntax of different programming languages
- **Reference**: Quick reference for developers exploring new languages
- **Fun**: Discover esoteric and unusual programming languages
- **Historical**: See how programming languages have evolved over time

## 🚀 How to Run

Each file contains a "Hello, World!" program. The method to run each depends on the language:

### Compiled Languages (C, C++, Rust, Go, etc.)
```bash
# Example for C
gcc hello.c -o hello
./hello
```

### Interpreted Languages (Python, Ruby, etc.)
```bash
# Example for Python
python hello.py
```

### JavaScript/Node.js
```bash
node hello.js
```

### Java
```bash
javac hello.java
java hello
```
